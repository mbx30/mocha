# Issue #229 — Optical Character Recognition (OCR)

**Goal:** Convert scanned PDFs and image-based PDFs into searchable, editable text.

**Design sketch:**
- Use Tesseract (local) or cloud Vision API as a configurable backend.
- Detect text-based vs scanned PDFs by checking for embedded fonts and text operators.
- Render scanned pages to images, run OCR, overlay hidden text layer on original PDF.
- Allow selected-page OCR.

**Prerequisites:**
- `tesseract` crate or external binary; or API key for cloud Vision.
- PDF fixture corpus with scanned samples.

**Implementation checklist:**
- [ ] Add OCR backend selection in settings.
- [ ] Implement `run_ocr(path, pages) -> OcrResult` command.
- [ ] Overlay OCR text as invisible text layer.
- [ ] Surface confidence scores in UI.
